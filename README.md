# Smart Bharat – AI-Powered Civic Companion 🇮🇳

## Overview

Smart Bharat is a Generative AI-powered civic platform designed to simplify citizen interactions with government services. The platform helps users discover relevant schemes, understand documentation requirements, report public issues, and receive personalized multilingual assistance through an intelligent AI companion.

Built for **PromptWar Hackathon**, Smart Bharat aims to make governance more accessible, transparent, and user-friendly for every citizen.

---

## Problem Statement

Citizens often face challenges such as:

* Difficulty finding the right government services
* Complex and hard-to-understand official information
* Confusion regarding required documents
* Language barriers
* Inefficient complaint reporting and tracking

Smart Bharat addresses these challenges using Generative AI.

---

## Key Features

### 🤖 AI Civic Companion

Ask questions in natural language and receive simplified, personalized guidance.

### 🗂️ Smart Service Discovery

Get recommendations for government schemes and services based on user needs.

### 📄 Document Assistant

Understand required documents, eligibility criteria, and application processes.

### 🌍 Multilingual Support

Interact with the platform in multiple Indian languages.

### 📸 AI-Powered Issue Reporting

Upload images of civic issues such as potholes, garbage, or broken streetlights and automatically generate structured complaints.

### 🛣️ Personalized Citizen Roadmap

Receive step-by-step guidance for completing government processes and applications.

### 🎯 Life Event Assistant

Get relevant services and schemes based on major life events such as education, employment, business registration, retirement, and more.

## Tech Stack

* Frontend: React / Next.js
* Backend: Node.js / Express
* AI: Gemini / OpenAI API
* Database: Supabase / PostgreSQL
* Authentication: Clerk / Firebase Auth
* Deployment: Vercel
